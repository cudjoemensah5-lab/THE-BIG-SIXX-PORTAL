# 🎓 The Big Six · UPSA Results Portal

A private academic results portal for **The Big Six Study Group** at the University of Professional Studies, Accra (UPSA).

## 📌 About
This portal allows the group admin (teacher) to enter and publish exam results, and students to privately view their own results using a secure token.

## 🔐 Access
- **Admin:** Protected by a 4-digit PIN (hashed & secured)
- **Students:** Log in with your Student ID and private token given by your teacher

## 🌐 Live Portal
👉 [Click here to access the portal](https://YOUR-USERNAME.github.io/big-six-portal/)

## ⚙️ Built With
- HTML, CSS, JavaScript
- Firebase Firestore (real-time cloud sync)
- Chart.js (analytics & charts)
- UPSA 4.0 CGPA Grading Scale
